## 👤 Об авторе
Я занимаюсь практической информационной безопасностью, исследованием уязвимостей и анализом открытых данных (OSINT). Разрабатываю автономный софт для аудита безопасности, автоматизации рутины и цифровой криминалистики. Мой стек ориентирован на скрытность, высокую производительность и работу без внешних облачных зависимостей.

## 🛠️ Технологический Стек

*   **Языки программирования:** Python (Скрипты, Автоматизация, Анализ), Bash, HTML5 / CSS3 (Интерфейсы утилит).
*   **Среда и ОС:** Kali Linux (Экспертный уровень), Архитектура Linux систем, Docker, Git.
*   **Профильные направления:** 
    *   **OSINT:** Поиск утечек, анализ метаданных, автоматизированный сбор цифровых следов.

    *   **Automation:** Написание парсеров, локальных движков и Telegram-ботов для кибербезопасности.

## 📦 Ключевые Проекты и Направления

Я не ограничиваюсь одной утилитой и развиваю инструменты под разные аналитические задачи:

### 🌐 1. Цифровая криминалистика и OSINT
*   **GeoHunter** — Локальная автономная лаборатория анализа изображений. Извлечение EXIF/GPS, расчет теней, поиск следов графических редакторов через математические фильтры пикселей и локальный OCR.
*   **grabcam / EXIF** — Проекты, ориентированные на исследование механизмов сбора метаданных и работы с камерами устройств в реальном времени.

### 🤖 2. Автоматизация и боты
*   **black-bot / v2** — Разработка защищенных модулей автоматизации для мессенджеров, администрирование систем через API и управление логами Kali Linux.

### 📚 3. Исследования и Обучение
*   **DARKWEB & Kali-Linux-Academy** — Репозитории с материалами, скриптами и аналитическими заметками по изучению теневого сегмента сети, защите данных и тестированию на проникновение.
