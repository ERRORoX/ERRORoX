Я независимый исследователь в сфере информационной безопасности, специализирующийся на анализе открытых данных (OSINT) и цифровой криминалистике (Forensics). *   **Языки программирования:** Python (автоматизация, парсинг, математические алгоритмы), Bash (скрипты для автоматизации в Linux).

## 📦 Текущие проекты (Open Source)

На данный момент я активно развиваю два ключевых направления:

### 🌐 1. Geohunter
Профессиональная автономная лаборатория для цифрового детектива. Скрипт предназначен для глубокого извлечения информации из фотографий в один клик.
*   **Функционал:** Чтение EXIF/GPS, калькулятор положения солнца для проверки теней, ELA-анализ сжатия пикселей, OCR улик (карты, криптокошельки, госномера).
*   **Статус:** В активной разработке. Интегрируются классические офлайн-алгоритмы геометрии лиц.

### 📸 2. grabcam
Инструмент, ориентированный на исследование механизмов работы с веб-камерами устройств и демонстрацию уязвимостей, связанных со сбором метаданных в реальном времени.
