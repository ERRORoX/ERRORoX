## 🛡️ About Me
Я независимый исследователь в сфере кибербезопасности, специализирующийся на **OSINT (разведке по открытым источникам)**, **цифровой криминалистике (Forensics)** и **обратной разработке (Reverse Engineering)**. Разрабатываю автономные и безопасные инструменты для анализа данных.

> *"Breaking systems to make them stronger. Every bug hides a hex."* 💻

---

## ⚡ Flagship Project: GeoHunter 🌐
**[GeoHunter](https://github.com)** — это профессиональная автономная лаборатория для цифрового детектива. 
*   **Локальный движок:** Никаких облачных API. Полная приватность ваших расследований.
*   **Анализ данных:** EXIF/GPS, солнечный калькулятор теней, ELA (криминалистика сжатия), интеллектуальный OCR улик и локальный геометрический анализ лиц.
*   **Безопасность:** Защищено строгой лицензией **GNU GPLv3**.
